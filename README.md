### Features
* `Generate large comments from input text`
* `Automatic language detection`
* `Command Palette`
* `Context menu`
* `YOU` + `KEYBOARD` + `SHORTCUT`

### Requirements

* Visual Studio Code
* Keyboard or mouse
* Hands
