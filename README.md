# Large Comments Generator

### Features
- [x] `Generate large comments from input text`
- [x] `Automatic language detection`
- [x] `Command Palette` 
- [x] `Context menu` 
- [x] <kbd>YOU</kbd> + <kbd>KEYBOARD</kbd> + <kbd>SHORTCUT</kbd>
### Requirements

* Visual Studio Code
* Keyboard or mouse
* Hands
